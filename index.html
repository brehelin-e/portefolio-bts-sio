Serveur de supervision Centreon

Contexte du projet

Installation et configuration d'un serveur Centreon pour la coopérative Terres de l'Ouest afin de centraliser la supervision des équipements et serveurs IT répartis sur plusieurs sites.

Objectifs

  • Superviser serveurs, routeurs, switchs et bornes Wi-Fi
  • Réagir rapidement aux alertes critiques (CPU, RAM, disques)
  • Créer des vues personnalisées pour les techniciens
  • Automatiser les alertes avec seuils prédéfinis

Réalisations techniques

  • Installation : Centreon 24.10 sur Debian 12 via SSH (MobaXterm)
  • Configuration réseau : IP fixe, ports 80/443, DNS et passerelle
  • Ajout des équipements : Configuration SNMP v2c pour tous les hôtes
  • Services supervisés : Memory (80% warning, 90% critical), CPU, Disques, Ping
  • Vues personnalisées : Dashboard RAM/Disques et État des hôtes
  • Automatisation : Déploiement SNMP via GPO sur tous les serveurs

Résultats

La coopérative dispose désormais d'un outil centralisé de supervision en temps réel affiché sur un écran dédié avec rotation automatique des vues toutes les 15 secondes. Réduction significative des temps de détection et résolution des incidents.

Centreon 24.10 Debian 12 SNMP v2c MobaXterm SSH MariaDB PHP 8.2

Optimisation SEO - Blog Immobilier

Problématique

Un blog immobilier actif depuis 2015 connaissait une chute drastique de positionnement et de trafic due à de nombreuses erreurs techniques SEO affectant l'indexation Google.

Audit initial

  • Score de santé : 51/100 (Ahrefs)
  • Normes protocolaires : 1750 URLs affectées (erreurs HTTP 404, 500, redirections 301/302)
  • Normes SEO : 951 URLs problématiques (doublons H1, URLs mal formées)
  • Normes implicites : 1222 URLs (images non optimisées, vitesse de chargement)

Solutions mises en œuvre

  • Optimisation images : Compression avec Imagify + conversion WebP
  • Correction URLs : Réécriture des slugs + redirections 301 appropriées
  • Réorganisation balisage : Correction H1 uniques et hiérarchie H2-H4 logique
  • Sécurité HTTPS : Installation plugin HSTS pour connexions sécurisées
  • Conformité RGPD : Mise en place des bonnes pratiques de protection données

Résultats finaux

  • Score de santé : 100/100 (+49 points)
  • Normes protocolaires : 17 URLs (-99%)
  • Normes SEO : 259 URLs (-73%)
  • Normes implicites : 435 URLs (-64%)

Impact : Redressement du positionnement SEO et amélioration de la visibilité dans les résultats de recherche Google.

Déploiement 200 Ordinateurs - CD29

Mission

Préparation et configuration de 200 ordinateurs portables Thinkpad T480 destinés aux assistants sociaux du Conseil Départemental du Finistère.

Procédure de déploiement

  1. Déballage : Réception et inventaire des 200 Thinkpad T480
  2. Installation réseau : Connexion RJ45 au réseau du CD29
  3. Boot PXE : Démarrage F9 et sélection "Déploiement CD29"
  4. Master Windows : Installation automatique de Windows 10 personnalisé avec logiciels préinstallés (Cisco, TeamViewer)
  5. Tests de validation : Checklist complète de 13 points par ordinateur
  6. Conditionnement : Pochette de transport + chargeur + câble

Tests de validation (par ordinateur)

  • ✓ 8 Go RAM minimum
  • ✓ Présence Cisco et TeamViewer
  • ✓ Caméra, micro et haut-parleurs fonctionnels
  • ✓ Test clavier complet (onlinemictest.com)
  • ✓ Ports USB-A et USB-C
  • ✓ Connectivité Wi-Fi
  • ✓ Test HDMI sur écran externe
  • ✓ Batterie et chargeur fonctionnels
  • ✓ Trackpad opérationnel
  • ✓ Nettoyage complet avant livraison

Compétences développées

  • Déploiement système à grande échelle
  • Création et gestion de masters Windows
  • Tests qualité et validation matériel
  • Documentation des procédures
  • Organisation et rigueur sur projet d'envergure

Réparations Smartphones Avancées

Expertise développée

Réparation de smartphones haut de gamme avec des techniques professionnelles d'encollage et de remplacement de composants. Plus de 30 réparations réalisées durant le stage chez Save Quimper.

Cas d'étude : Samsung Galaxy S23 Ultra

Type : Remplacement écran complet avec châssis (pièce d'origine constructeur)

  • Diagnostic : Écran cassé mais téléphone fonctionnel
  • Chauffe : 3-4 minutes pour ramollir la colle
  • Démontage complet : Retrait méthodique de tous les composants
  • Composants manipulés : Carte mère, caméras, batterie, système à induction, stylet, boutons
  • Remontage : Installation nouveau châssis + adhésif étanchéité
  • Tests : Validation tactile via appel d'urgence avant recollage
  • Garantie : 1 an hors casse et oxydation

Cas d'étude : Xiaomi P30 Lite (Encollage)

Type : Encollage LCD (solution économique)

  • Test LCD : Vérification nouveau composant avant installation
  • Retrait écran cassé : Plaque métallique ultra-fine
  • Préparation châssis : Nettoyage résidus colle + verre
  • Application colle B7000 : Tour complet du châssis
  • Séchage : 10-20 min avant pose + pinces de maintien 1h30
  • Tests validation : Tactile + affichage fonctionnels

Matériel utilisé

  • Machine de chauffe Samsung
  • Spatules fines et médiators
  • Tournevis de précision
  • Colle B7000 professionnelle
  • Tapis antistatique relié à la terre
  • Pinces de maintien
  • Produits nettoyants spécifiques

Compétences acquises

  • Minutie et précision dans les manipulations
  • Gestion de composants fragiles
  • Diagnostic et tests de validation
  • Relation client et conseil
  • Gestion du stock et traçabilité
